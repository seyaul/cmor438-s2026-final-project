# Repository For Rice CMOR 438: Data Science and Machine Learning
The goal of this repository is to create a python package that implements various supervised and unsupervised machine learning algorithms.
The dataset used is GuitarSet: Q. Xi, R. Bittner, J. Pauwels, X. Ye, and J. P. Bello, "​Guitarset: A Dataset for Guitar Transcription", in 19th International Society for Music Information Retrieval Conference, Paris, France, Sept. 2018.

We will demonstrate how we can use the various machine learning methodologies to identify monophonic signals in the realm of Guitar. We will attempt to extend it to polyphonic signals.
